The control-plane API is the tenant-scoped API used by the dashboard, CLI, and SDKs.

Use it for:

- Agent profile management.
- Run creation, streaming, cancellation, and replay.
- Session lookup.
- Skills, MCP servers, memory, storage, secrets, and pools.
- API keys, published-agent configuration, billing, usage, stats, and workflows.

## Authentication

Send a tenant API key as a bearer token:

```bash
Authorization: Bearer ao_...
```

When your deployment uses explicit tenant selection, also send:

```bash
X-Tenant-ID: acme
```
